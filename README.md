# 🎁 A Gift For You

"A Gift For You" is an interactive web experience designed as a personal digital gift.

Visitors answer a series of questions, and their choices lead them to different love notes and scenarios, creating a personalized experience each time.

The project explores how web interactivity can be used to express emotional storytelling.

---

## ✨ Features

- Branching interactive paths
- Personalized responses based on choices
- Multiple possible outcomes
- Narrative-style interaction

---

## 🛠 Tech Stack

- EJS

Backend
